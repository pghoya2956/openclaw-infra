## 스킬 설치

### CLI 설치

```bash
npx clawhub@latest install <skill-slug>
```

### 수동 설치

스킬 복사 위치:
- 글로벌: `~/.openclaw/skills/`
- 워크스페이스: `<project>/skills/`

우선순위: Workspace > Local > Bundled
